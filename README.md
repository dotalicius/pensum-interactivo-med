# Pensum Interactivo

Aplicación web para estudiantes de Medicina de la UNLP que permite visualizar y gestionar el progreso académico del pensum de la carrera.

## Qué hace la aplicación

El Pensum Interactivo te permite:

- Ver todas las materias del plan de estudios organizadas por año
- Marcar materias como regulares o aprobadas
- Visualizar correlatividades entre materias
- Llevar un registro de notas y fechas de exámenes
- Contar horas optativas para acceder a la PFO
- Seguir tu progreso general en la carrera

## Cómo usar la aplicación

### Ver el pensum
- Las materias se muestran organizadas por año académico
- Cada materia muestra su régimen (anual, cuatrimestral) y carga horaria
- El color de la materia indica su estado: pendiente, regular o aprobada

### Actualizar estado de materias
- Haz clic en una materia para abrir el modal de detalles
- Cambia el estado de "Pendiente" a "Regular" o "Aprobado"
- La aplicación verifica automáticamente las correlatividades

### Gestionar detalles
- Agrega información de comisión, profesor y horarios
- Registra fechas de exámenes y notas
- Guarda enlaces a materiales o documentos

### Seguir tu progreso
- El dashboard muestra estadísticas en tiempo real
- Contador de materias aprobadas, regulares y pendientes
- Progreso de horas optativas para la PFO
- Promedio general de notas

## Requisitos del sistema

- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- No requiere instalación ni registro

## Datos y privacidad

- Toda la información se guarda localmente en tu navegador
- No se envían datos a servidores externos
- Los datos persisten entre sesiones hasta que limpies el navegador
